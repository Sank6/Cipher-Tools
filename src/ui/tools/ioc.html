Finding the Keylenght

This IOC graph is useful for finding the keylength Vigeneres, Beauforts and other similar polyalphabetic ciphers. The way this graph above works is by spliting the text into n columns and checking the index of coincidence for each row. These IOCs for each column are averaged. The columns where the IOCs are high, are shown as spikes on the graph and these are all multiples of the keylength.

Finding the Key

Once you have the keylength, you need the key. You can find the each letter of the key by doing a frequency analysis on each column and treat the columns like a Caeser shift. Then, once you have the whole keyword, you can just decode it.
