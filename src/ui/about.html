About

Nickel is an open-sourced cipher tool to automate decryption of some ciphers, built on Electron and a some web-based packages: jQuery, FontAwesome, Chart.js, Noty, jQuery Modal, Google Fonts. The colour palette is inspired from Nord Theme

Caesar

The Caesar auto solver uses a brute-force method as there are only 26 possibilities. For every possible shift, the actual frequency of the letter is compared with the expected frequency and then sum all the differences in these two percentages. The shift with the smallest difference is assumed to be the "solution" and so the key is the shift.

Vigenere

The Vigenere auto solver has two steps. The first is to get the key length through the Index of Coincidence method and comparing the factors of the columns with the top 3 index of coincidence. To get the key, each column is treated as a separate Caesar cipher and the above method is used to get each letter of the key.
